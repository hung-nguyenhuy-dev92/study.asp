Controller Action Return Type & HTTP Status Code: 
https://learn.microsoft.com/en-us/aspnet/core/web-api/action-return-types?view=aspnetcore-5.0
1xx: informational response
2xx: seccess
- 200 Ok - request has seccessded
- 201 Created - a new resource has been created as a result (dành cho nhưng api tạo sẽ trả về cho 1 object vừa tạo)
- 204 No content - No need to return a response body (thường dùng cho api update sẽ trả về cho là no content) NoContent()
lưu ý có thể gộp 201 hoặc 204 thành 200 cũng được
3xx: redirection
4xx: client error
- 400 Bed Request:  the request could not be understood by server due to malformed syntax (server không hiểu nhưng gì client gửi lên)
- 401 Unauthorized: Request requires user authentication (cần phải chứng thực người dùng nghĩa là mình cần phải đăng nhập)
- 403 Forbidden: Unauthorized request (xác mình người dùng xong rồi nhưng có quyền (authorized) trên api)
- 404 Not Found: (tìm kiếm lấy theo id mà ko có id đó)
- 405 Method not allowed (mothod bị lộn method là post nhưng mình lại request là get)
- 422 Unprocessable entity (trong quá trình post put thường request object lên body nhưng object bị thiếu hoặc sai cú pháp)
5xx: server error
500
503
504
